annis.provider.SaltProjectProvider.NonCloseableInputStream Class Reference

Detailed Description

An InputStream that delegates all of the actions to a delegate object but can't be closed.

In SaltProjectProvider#readFrom(java.lang.Class, java.lang.reflect.Type, java.lang.annotation.Annotation[], javax.ws.rs.core.MediaType, javax.ws.rs.core.MultivaluedMap, java.io.InputStream) we have to make sure the provided InputStream is not closed. Unfurtunally the SAX parser will always close the stream and there seems to be no option to avoid that. Thus we use this hack where the close() function does nothing.

Fetches the meta data for a top level corpus.

If the path path is selected, all annotations from the corpora and subcorpora of the name path are included.

Path(s)

  1. GET annis/meta/doc/{top}
  2. GET annis/meta/doc/{top}/{doc}
  3. GET annis/meta/doc/{top}/{doc}/path

MIME

accept: application/xml application/json

Parameters
topLevelCorpusDetermines the corpus, for which the annotations are retrieved.
docnameDetermines the document name, for which the annotations are fetched. If null, all annotations of all documents are fetched.
pathif true, the annotations of all corpora and documents contained in the path are also included
Returns
The xml representation of a list wich contains Annotation objects.

annis.security.BasicAuthOrAnonymousFilter Class Reference

Detailed Description

Authentificate with a static anonymous user account when no real authentification was given.

If there is an authentification given this class will act like it's parent class. The default value for password and user name are "anonymous". You can change them with the anonymousUser and anonymousPassword properties.

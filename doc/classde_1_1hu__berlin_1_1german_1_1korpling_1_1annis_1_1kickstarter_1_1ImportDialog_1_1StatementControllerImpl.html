void de.hu_berlin.german.korpling.annis.kickstarter.ImportDialog.StatementControllerImpl.cancelStatements ( )

Interrupts a sql statement via the JDBC-Driver.

It relies on the actual implementation of the JDBC-Driver, if this method has an effect.

If the PreparedStatement#cancel() It also set the internal isCancelled-flat to true, so isCancelled() will always return true. This behaviour is inspired by the method interrupt method of the Thread#interrupt() method. The implementation of the AdministrationDao interface should poll against this flag and do not execute further sql-statements.

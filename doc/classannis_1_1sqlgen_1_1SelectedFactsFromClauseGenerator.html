annis.sqlgen.SelectedFactsFromClauseGenerator Class Reference

Detailed Description

A simple FromClauseSqlGenerator that statically returns the name of the view which contains the selected facts table.

In order to work this needs the SelectedCorporaWithClauseGenerator to be enabled and we have to use the facts table. This implementation is not prepared for a dynamic scheme where node, rank etc. are seperate tables.
